Инструкция по использованию сайта для поиска и сохранения вакансий:

1. Открытие сайта:
    Перейдите на главную страницу вашего сайта.

2. Поиск вакансий:
    Введите ключевое слово в поле для поиска вакансий. Например, "разработчик", "менеджер" и т.д.
    При необходимости, укажите минимальную зарплату и/или требуемый опыт работы, если такие опции доступны.

3. Запуск поиска:
    Нажмите кнопку "Поиск" или аналогичную, чтобы начать поиск вакансий.

4. Просмотр результатов:
    На странице будут отображены найденные вакансии. Вы можете просмотреть их заголовки, компании, зарплаты и ссылки для получения более подробной информации.

5. Завершение работы:
    Когда вы завершите работу с сайтом, просто закройте вкладку браузера или вернитесь на главную страницу для выполнения новых поисковых запросов.
